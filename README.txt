HOW TO DEPLOY ON GITHUB PAGES (Free Hosting):

1. Create a new public repository on GitHub (e.g., love-at-first-bite-menu).
2. Upload `index.html` and optionally your `logo.png` using the "Add file → Upload files" button.
3. Commit the changes.
4. Go to "Settings" → "Pages" → Under "Source", select "Deploy from a branch".
5. Choose the "main" branch and folder "/ (root)" and click Save.
6. Wait ~1 minute. Your website will be live at:
   https://yourusername.github.io/reponame/

You can then point your QR code to that link.
